## Project Overview

This repository contains my **COS106 – Introduction to Web Technologies** practical term project submitted at **MIVA Open University**.

The project demonstrates the use of **HTML5**, **CSS3**, and **Vanilla JavaScript** to develop a responsive multi-page student portfolio and academic management website. It showcases my academic profile, technical skills, sample projects, and an interactive planner designed to help students organize their academic activities.
