# COS106 Student Portfolio

This project is a responsive student portfolio and academic management website for Andrew Moses. It includes:

- A home page with a profile section, biography and featured projects.
- An about page with academic and personal information.
- A projects page showing three practical web development projects.
- An academic planner with add, complete and delete task features.
- A contact page with JavaScript form validation.

## Files
- index.html
- about.html
- projects.html
- planner.html
- contact.html
- css/style.css
- js/main.js
- js/planner.js
- js/contact.js
- images/
- media/

## How to view
Open index.html in a browser, or use a simple local development server if preferred.
